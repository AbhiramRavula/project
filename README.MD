# Community Share App

Welcome to the Community Share App! This is a fully-featured web application built with React, TypeScript, and Tailwind CSS, designed to facilitate sharing posts and services within a community. The app leverages Zustand for state management and integrates with Web3 for blockchain functionalities.

## Features

- **User Authentication**: Users can log in and manage their sessions.
- **Post Sharing**: Users can create and view posts.
- **Service Listings**: Users can offer and browse services.
- **Marketplace**: A dedicated section for marketplace activities.
- **Fact Checker**: A feature to verify articles and information.
- **Responsive Design**: The app is designed to be fully responsive and user-friendly.

## Technologies Used

- **Frontend**: 
  - React
  - TypeScript
  - Tailwind CSS
  - Zustand (for state management)
  - Lucide React (for icons)
  
- **Backend**: 
  - Axios (for API calls)
  - Web3 (for blockchain interactions)

## Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/community-share-app.git
   cd community-share-app
   ```

2. Install the dependencies:

   ```bash
   npm install
   # or
   yarn install


   instal react react-dom
   ```

3. Set up your environment variables (if needed).

### Running the Application

To start the development server, run:
 ```bash
 npm run dev 
 ```
